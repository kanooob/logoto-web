Logoto Help - Automate your Logo & Name! 🇬🇧

I am the bot specialized in automating the change of your server's logo AND name, without requiring complex configurations or lines of code after the initial setup.

Commands

  • /setup : First command to run. Automatically creates the required category and channels (Logoto, log-logoto) for a quick start.
  • /add-an-event : Automatically creates a scheduling channel with the correct format for your future events (Logo or Name).
  • /help : Displays the bot commands list (or this documentation page!).
  • /info : Returns all useful information regarding the current state and status of the bot.
  • /ping : Get the real-time latency between the bot and the Discord API.
  • /invite : Get the official invitation link to add Logoto to your own server.
  • /support : Join our support server to ask questions or get assistance.

Automatic Logo Change System

The bot monitors specific text channels to schedule and execute server avatar changes. You can use the /add-an-event command or create them manually:

  1. Create the Scheduling Channel (Logo):
    • The channel name must strictly follow this format: l-[DAY]-[MONTH] (or generated via /add-an-event).
    • EXAMPLE: To schedule an automatic logo for Halloween (October 31st): l-31-10
  2. Prepare the Logo:
    • Upload your desired image to any channel on your server, right-click it, and copy its direct link (URL).
    • The link must point to a valid image format (PNG, JPEG).
  3. Schedule the Change:
    • Edit the Channel Topic of the text channel created in step 1.
    • Paste the direct image link directly into the topic of this channel.
  4. Result:
    • The bot will automatically change the server's profile picture on the specified day and month!

Automatic Name Change System

The principle is identical for automatically changing your Discord server's display name on a key date:

  1. Create the Scheduling Channel (Name):
    • The text channel name must follow this format: n-[DAY]-[MONTH] (or generated via /add-an-event).
    • EXAMPLE: To schedule a name change on January 1st: n-1-1
  2. Schedule the Change:
    • Edit the Channel Topic of the created channel.
    • Type the exact new full name that the server should take inside the channel topic.
  3. Result:
    • The bot will automatically apply the new name on the specified date!
IMPORTANT NOTE: For the automatic daily daily verifications and changes to run properly, the channel named log-logoto must absolutely exist on your server.

The bot uses this channel to send its daily technical loading messages (l-Loading or n-Loading), which trigger the server configurations refresh. The /setup command takes care of creating it cleanly for you.
