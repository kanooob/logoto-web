Aide et Commandes Logoto : Automatisez votre Logo & Nom Discord 🇫🇷

Je suis le bot spécialisé dans l'automatisation du changement de logo et du nom de votre serveur Discord, sans nécessiter de configurations ou de lignes de code complexes.

Les Commandes du Bot Logoto

  • /setup : Première commande à faire. Crée automatiquement la catégorie et les salons nécessaires (Logoto, log-logoto) pour un démarrage rapide.
  • /add-an-event : Crée automatiquement un salon de planification au bon format pour vos événements futurs (Logo ou Nom).
  • /help : Affiche la liste des commandes du bot (ou cette page de documentation !).
  • /info : Renvoie toutes les informations utiles concernant l'état actuel et le statut du bot.
  • /ping : Obtenez la latence en temps réel entre le bot et l'API Discord.
  • /invite : Obtenez le lien d'invitation officiel pour ajouter Logoto sur votre propre serveur.
  • /support : Rejoignez notre serveur de support pour poser vos questions ou obtenir de l'aide.

Système de Changement de Logo Automatique

Le bot surveille la présence de salons textuels spécifiques pour planifier et exécuter les changements d'avatar du serveur. Vous pouvez utiliser la commande /add-an-event ou faire la création manuellement :

  1. Créez le Salon de Planification (Logo) :
    • Le salon doit suivre scrupuleusement le format : l-[JOUR]-[MOIS] (ou généré via /add-an-event).
    • EXEMPLE : Pour programmer un logo automatique pour Halloween (31 octobre) : l-31-10
  2. Préparez le Logo :
    • Envoyez l'image souhaitée dans n'importe quel salon de votre serveur, faites un clic droit dessus et copiez son lien direct (URL).
    • Le lien doit pointer vers un format d'image valide (PNG, JPEG).
  3. Planifiez le Changement :
    • Modifiez le Sujet du Salon (Channel Topic) du salon textuel créé à l'étape 1.
    • Collez le lien direct de l'image directement dans le sujet de ce salon.
  4. Résultat :
    • Le bot changera automatiquement la photo de profil du serveur au jour et au mois indiqués !

Système de Changement de Nom Automatique

Le principe est identique pour modifier automatiquement le nom d'affichage de votre serveur Discord à une date précise :

  1. Créez le Salon de Planification (Nom) :
    • Le salon doit suivre le format : n-[JOUR]-[MOIS] (ou généré via /add-an-event).
    • EXEMPLE : Pour planifier un changement de nom le 1er Janvier : n-1-1
  2. Planifiez le Changement :
    • Modifiez le Sujet du Salon (Channel Topic) du salon créé.
    • Écrivez textuellement le nouveau nom complet que devra prendre le serveur dans le sujet.
  3. Résultat :
    • Le bot appliquera automatiquement le nouveau nom au jour et au mois spécifiés !
NOTE IMPORTANTE : Pour que les vérifications et les changements automatiques s'exécutent correctement, le salon nommé log-logoto doit impérativement exister sur votre serveur.

Le bot utilise ce salon pour envoyer ses messages techniques quotidiens de chargement (l-Loading ou n-Loading), ce qui déclenche la mise à jour des configurations du serveur. La commande /setup s'occupe de le créer proprement pour vous.
